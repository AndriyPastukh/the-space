# the-space

# Про проєкт

TheSpace — простір досвіду, підтримки та перших кроків в професійному розвитку.
Це спільнота для новачків яка допомагає отримати перший практичний досвід через, виконання невеликих завдань-проектів, обмін знаннями і досвідом, взаємодію з іншими учасниками спільноти.

У TheSpace можна:
* виконувати невеликі завдання-проекти
* отримати перший практичний досвід
* обмінюватися знаннями
* будувати власне портфоліо 
* знаходити та створювати спільноти, команди
