# Run Frontend Locally

## 1. **Open a terminal and navigate to the frontend folder**

```bash
C:\...\project-the-space> cd frontend
```

_Skip this step if you are already in the `frontend` directory._

## 2. **Install dependencies**

```bash
npm install
```

## 3. **Run**

```bash
npm run dev
```

## 4. **Open the app in browser**

```bash
http://localhost:5173
```

_If the port is busy, use the one shown on your terminal._
